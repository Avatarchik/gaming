Amaya Casino 5 Libraries - Release Notes

Overview

Platform Library: The Amaya HTML5 Integration Library is a collection of classes to allow sending of information to the Casino 5 framework for systems which are integrating it. The main class is amaya.PlatformServices which is used to send messages to the framework (via Post Message events).

Version 1.3

New Features
  • Added "getGameLaunchURL" method to PlatformServices. This method is used to generate the source url of the iFrame that should display the game.
  • Added "enableSound" and "disableSound" methods to PlatformServices.
  • Added amaya.platform.constants.Account enumeration to define constants associated with accounts.
  • Added amaya.platform.constants.ResumeType enumeration to define constants used to define how to handle resuming unfinished games.
  • Added amaya.platform.constants.DeviceType enumeration to define constants associated with device types.
  • Added amaya.platform.constants.OptionId enumeration to define constants associated with options.

Version 1.2

New Features
  • Changed the name of the minified js file (and its corresponding docs zip) to html5integration.|version|.min.js

Version 1.1

New Features
  • Added showPaytable method to PlatformServices
  • Added new state notification ids for "STATE_BONUS_GAME_START" and "STATE_BONUS_GAME_END"
  • Added new event constants for "GAME_READY", "AUTO_PLAY_START" and "AUTO_PLAY_END"

Version 1.0

New Features
  • Initial version of the library used to integrate the HTML5 product into a host.
