Amaya Casino 5 Libraries - Release Notes

Overview

API Library: The Amaya Casino 5 API Library is a collection of classes designed to allow a game to interact with the Casino 5 Framework. The main class is amaya.GameServices which is used to both send and receive messages to the framework (via public methods and event listeners respectively) as well as query the state of many runtime properties (such as the state of options, session values etc).

PMAPI Library: The Amaya Casino 5 PMAPI Library is a collection of classes designed to allow a game to interact with the Casino 5 Framework using the browser's PostMessage functionality. The main class is amaya.GameServicesPM which is used to both send and receive messages to the framework (via public methods and event listeners respectively) as well as query the state of many runtime properties (such as the state of options, session values etc).

SDK Library: The Amaya Casino 5 Core SDK Library is a collection of classes designed to make the development of games for the Casino 5 Client Platform easier. It contains partial implementations of games and messages that can be extended as well as utility classes for performing functions such as formatting monetary amounts, working with JSON, managing canvases etc.

Integration Library: The Amaya Casino 5 Integration Library is a collection of classes to provide information to systems which are integrating the Casino 5 platform. The main class is amaya.IntegrationServices which is used to receive messages from the framework (via Post Message events).
